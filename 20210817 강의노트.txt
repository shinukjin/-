	: CHECK OR NOT NULL : 저장 가능한 데이터 값의 범위나 조건을 지정해 설정한 값만 허용

11. DML, DDL, TCL, DCL ?? ORACLE_기초 13쪽
	DML : Data Manipulation Language : INSERT INTO 테이블명
									 : UPDATE 테이블명
									 : DELETE 테이블명
									 >> 데이터 변경 시 사용한다.

	DDL : Data Definition Language	 : CREATE TABLE 테이블명
									 : DROP TABLE 테이블명
									 : ALTER TABLE 테이블명 
									 >> Object 생성과 변경 시 사용.
	TCL : Transaction Control Language : COMMIT, ROLLBACK, SAVEPOINT : Transaction 종료 및 취소
	DCL : Data Control Language : GRANT, REVOKE 권한 부여 및 ㅟ소

12. TABLE, ROW, COULMN, PIMARY, FIELD, NULL

	TABLE : 관계형 데이터베이스의 기본 데이터 저장 구조
		  : 하나의 테이블은 여러개의 ROW로 구성
	
	ROW : 하나의 테이블을 구성하는 다른 유형의 데이터
	COLUMN : 하나의 ROW를 구성하는 구성요소
	PRIMARY : 테이블에서 각 ROW를 유일하게 구분하는 COLUMN KEY
	FIELD : 테이블에서 ROW와 COLUMN이 교차하는 데이터
	NULL : 데이터가 존재하지 않는 COLUMN의 값
